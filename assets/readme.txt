Mobile Api and Content Only function for app deployment of JustAwesome Webdesign

== Description ==
Content Only function:
- Hides Footer and Header when ?native parameter is added to the url
- Shows Footer and Header again when ?nonative parameter is added

Creates public api for products and categories:
/wp-json/justawesome-woocommerce2app/v1/products
/wp-json/justawesome-woocommerce2app/v1/categories

3. Ask JustAwesome Webdesign for your personal App
